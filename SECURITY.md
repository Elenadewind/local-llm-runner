## Security Policy

### Reporting a Vulnerability
If you find a security issue, please report it privately to:  
**Email:** rusluvlang@yandex.ru  

Do NOT post security issues in public Issues.

### Supported Versions
We only provide security updates for the latest release.

### Dependencies
All dependencies are pinned in `requirements.txt`. We monitor for CVEs via:  
- GitHub Dependabot  
- Manual audits every 3 months

### Data Privacy
- No data is sent to external servers.  
- All processing happens locally.  
- The API does not log prompts or outputs by default.
